# 贡献指南

感谢你对贝塔眼镜 AR 虚拟试戴项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告 Bug

1. 在提交 Bug 之前，请先搜索是否已有相关的 Issue
2. 使用 Bug 报告模板创建新的 Issue
3. 提供详细的复现步骤和环境信息

### 提交功能请求

1. 在提交之前，请先搜索是否已有相关的功能请求
2. 使用功能请求模板创建新的 Issue
3. 详细描述你的需求和使用场景

### 提交代码

1. Fork 本仓库
2. 创建你的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m '添加某个特性'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

## 开发指南

### 环境要求

- Node.js 18+
- npm 9+ 或 pnpm 8+

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/your-username/beita-glasses.git
cd beita-glasses

# 安装依赖
cd reactViteIntegrationDemo
npm install

# 启动开发服务器
npm run dev
```

### 代码规范

- 使用 ESLint 进行代码检查
- 遵循现有的代码风格
- 添加必要的注释
- 提交信息使用中文，格式清晰
